# Bondr

**Lightweight SaaS for Small Glue/Adhesive Stick Manufacturers**

Bondr helps tiny factories (like my uncle's in Riyadh) track everything from raw bead/pellet inventory to finished glue stick production—without expensive ERP bloat.

### Why Bondr?
- Tailored for small-scale producers using spreadsheets or paper.
- Focus: bead stock, batch logging, defects/yield, basic costing & alerts.
- Privacy-first & self-hosted option (no forced cloud).
- Built for real users first.

### MVP Goals
- Bead/pellet inventory (lots, expiry, low-stock alerts)
- Batch production tracking (melt/extrusion settings, output, waste)
- Finished goods inventory
- Simple dashboard/reports

### Tech Stack (planned)
- Backend: Java 21 + Spring Boot 3.x
- Database: PostgreSQL
- Frontend: React (or React Native if mobile-first)
- Deployment: Docker + Render/Railway/VPS

Status: Early setup – MVP in progress.

Contributions, feedback, or beta interest welcome! Especially from adhesive/small manufacturing folks.
